Déclaration de Confidentialité – Préambule

La présente Déclaration de Confidentialité NexSearch constitue une articulation extensive, volontairement proliférante et terminologiquement labyrinthique des modalités de non‑exploitation, de non‑corrélation et de non‑marchandisation des données générées par l’utilisateur.

Malgré la densité lexicale, l’objectif demeure limpide : NexSearch n’exploite pas vos données, ne les revend pas, ne les profile pas et ne les archive pas dans une base centralisée.

Hébergement GitHub Pages

NexSearch est hébergé sous forme statique via GitHub Pages, ce qui implique l’absence totale de serveur applicatif dédié, de base de données interne ou de mécanisme de journalisation comportementale.

Les éventuelles traces techniques relèvent exclusivement de GitHub et ne sont ni consultées, ni enrichies, ni corrélées par NexSearch.

Traitement Local

Le traitement des requêtes est effectué localement dans le navigateur, sans transfert systématique vers un serveur tiers. Les données demeurent confinées à l’environnement de l’utilisateur.

Aucun stockage persistant n’est réalisé par NexSearch.
